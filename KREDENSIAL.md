# Kredensial Aplikasi Rumah Kos

## Cara Menjalankan

1. Copy folder project ke C:\xampp\htdocs\kos rumah
2. Buat database db_rumah_kos di phpMyAdmin
3. Edit file .env sesuai database Anda
4. Akses: http://localhost:8080/
